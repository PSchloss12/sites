Signature Verification Siamese Network

I crafted a custom siamese network architecture using PyTorch and trained, validated, and tested on a dataset from Kaggle with a partner. The finished model achieved a validation accuracy of 90% and test accuracy of 82.5% on signatures from the dataset.


Heap Management

Working in C, I built a set of custom memory management functions including malloc(), calloc(), realloc(), and free() with one partner. We learned how to inject them with custom linking to applications like cat.


Story Telling LSTM

Over the course of a semester, I researched and created a LSTM designed to help young authors solve writers' block by prompting the next line of text in Python using PyTorch and a custom training dataset. After creating a class for my LSTM model and functions for tokenizing data and creating a vocab set, I parsed over 40 texts from Project Gutenburg to use as training, testing, and validation data. I also created front end with HTML, CSS, and Flask utilizing this model in the background.


Unix grep and sed Variants

In Theory of Computing, I wrote a Pythonic, NFA-based, grep-style regular expression matcher with back-referencing capabilities that runs in linear time with a partner. Using this code as a basis, we then implemented sed's s command for regular expressions with replacement and its b command for conditional branching. We utilized GitHub throughout this project.


Turing Machine Compiler

In the same class, my partner and I wrote a tm_to_sed application which takes in the encoding of a turing machine in a text file and outputs a sed script equivalent to the tm file.


Reddit Browser

Before the Reddit API policy changes, I completed this full stack project to essentially recreate Reddit's functionality. I stored over one thousand subreddits in a database and links to these subreddits populated the home page. Each subreddit link would take the user to a posts page containing the latest data scrapped from Reddit. Each post could then be clicked to recursively populate the comments page with the comments for that post. Each of the above pages could be modified on a settings page to sort, filter, and limit the content.


Scheduler Design
